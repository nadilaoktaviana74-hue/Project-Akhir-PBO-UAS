## 📖 Deskripsi

**Manajemen Tugas PBO** merupakan aplikasi sederhana yang dikembangkan sebagai studi kasus pada mata kuliah **Pemrograman Berorientasi Objek (PBO)**. Project ini menerapkan konsep **Object-Oriented Programming (OOP)** menggunakan bahasa **Python**.

Aplikasi ini membantu pengguna dalam mengelola daftar tugas, mulai dari menambahkan tugas, memperbarui informasi tugas, menghapus tugas, hingga menandai tugas sebagai selesai.

- 🐍 Python
- 📓 Google Colab
